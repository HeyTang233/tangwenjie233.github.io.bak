Node.js官网下载安装并配置Node.js

按照教程安装npm

你需要

有一个github账号，没有的话去注册一个；
安装了node.js、npm，并了解相关基础知识；
安装了git for windows（或者其它git客户端）

在Github上新建一个仓库

仓库名格式 账户名.github.io
我的绑定了自己的域名不用在意

在建好的项目右侧有个settings按钮，点击它，向下拉到GitHub Pages，你会看到那边有个网址，访问它，你将会惊奇的发现该项目已经被部署到网络上，能够通过外网来访问它。
